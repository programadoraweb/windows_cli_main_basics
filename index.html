Windows Basics Command Line Interface Commands
Command Function
windows + R run e.g. open cmd
windows + E explorer
windows + Q query e.g. search progras
windows + G open Xbox. Followed by windows + Alt + R. you can record
cmd open command prompt
code . open visual studio code
dir will list all directories where you are
cd change directory
cd.. returns 1 level directory
title giveATitle give your command line a title
ctrl+c will spop any process that is being executed
cls clean screen
exit closes console
mkdir make a directory
rmdir remove a directory
type nul>fileName.fileExtension create a new file
echo .> fileName.fileExtension create a new file
echo hi there > fileName.fileExtension create a new file with content inside. This one says "Hi There"
ren fileName.fileExtension newName.fileExtension rename a file
del fileName.fileExtension delete a file
tree see the directory tree on your project
fc firstFileName.fileExtension secondFileName.fileExtension compare files
cd/d D: switches to drive D
help will list all available commands
F7 list all commands that we have executed so far
systeminfo get to see all system info, such as processor, windows version, ram, etc.
systeminfo >fileName.fileExtension export all system info to a file.
ipconfig/all under Ethernet you can get to se IP data.
ipconfig/all > filename.fileExtension export ip data to a file.
nslookup www.yourweb.com get to see the web's ip direction.
ping www.yourweb.com if you get a positive response the domain is working properly. We might me facing internet issues ourselves.
